@switch (view) { @case ('ngx-aside-ease') {

This library offers a lightweight, performant, and responsive aside, while leaving the styling control to the user. This library multiple options and offers a good user experience.

} @case ('preferences') {

User preferences are kept and stored in localStorage. Resize the aside panel and reload the page to see effects.

} @case ('performant') {

Lighweight library, defaulted to ChangeDetection.onPush. DRY patterns are followed. Resize event adding responsiveness is debounced. Implement an aside panel in your application in no time!

} @case ('responsive') {

This library is responsive, at the breakpoint you want. Resize the window to see effects. The AsideService offers a method to toggle the aside panel visibility, partially or totally.
Calling this method will either trigger a partial or full width panel, depending on the breakpoint set.

} @case ('controlable') {

Users have full control on aside panel visibility. Control a total show/hide, enable resize through touch and mouse control, or click on the resize arrow.

Don't hesitate to override the default styles classes (inspect the DOM to find the corresponding classes) with your creativity. Classes have been prefixed with 'ngx-*'. Set view encapsulation to 'none' in your hosting component.

} }
